# Python Projects - freeCodeCamp Certification

This repository contains my completed projects for the freeCodeCamp Python certification, covering OOP, data structures, and algorithms.

## Projects Included

### 1. Budget App
- **Key Concepts:** OOP, List Manipulation, Formatting.
- **Run:** `python A-Budget-App.py`

### 2. Custom Hash Table
- **Key Concepts:** Data Structures, Hashing Algorithms, Collision Handling.
- **Run:** `python A-Hash-Table.py`

### 3. Polygon Area Calculator
- **Key Concepts:** Class Inheritance, Geometry Logic, and UI Representation.
- **Description:** A program that uses a `Rectangle` base class and a `Square` subclass to calculate area, perimeter, and diagonal length, featuring a text-based visual representation of the shapes.
- **Run:** `python Polygon-Area-Calculator.py`

### 4. User Configuration Manager
- **Key Concepts:** CRUD Operations, Dictionary Management, and Data Normalization.
- **Description:** A utility script for managing user settings that handles adding, updating, and deleting pairs while ensuring case-insensitive key consistency.
- **Run:** `python User-Config-Manager.py`

### 5. Tower of Hanoi Solver
- **Key Concepts:** Recursive Algorithms and State Tracking.
- **Description:** An algorithmic solver for the Tower of Hanoi puzzle that uses recursion to move disks between rods and returns a log of every state change during the process.
- **Run:** `python Tower-of-Hanoi.py`
